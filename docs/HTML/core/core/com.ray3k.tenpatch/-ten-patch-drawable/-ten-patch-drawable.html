Creates a TenPatchDrawable. All stretch values must be defined in pairs.

Parameters

horizontalStretchAreas

Pairs of values defining stretch areas from the left of the graphic in ascending order in pixels. All values are inclusive with 0 being the left-most pixel (i.e. (0,2) defines a 3 pixel wide stretch area on the left of the graphic).

verticalStretchAreas

Pairs of values defining stretch areas from the bottom of the graphic in ascending order in pixels. All values are inclusive with 0 being the bottom-most pixel (i.e. (0,2) defines a 3 pixel high stretch area on the bottom of the graphic).
