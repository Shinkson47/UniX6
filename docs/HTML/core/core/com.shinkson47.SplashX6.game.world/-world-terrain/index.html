class WorldTerrain(width: Int, height: Int) : TiledMap, PartiallySerializable
Content copied to clipboard

The terrain for a game world.

Does not contain unit etc that are in the world.

Container for terrain and terrain navigation data in both raw Tile form (For generation, interpretation, and serialization) and TiledMap / TiledMapTileLayer form (for game engine & rendering).

n.b : TiledMap and it's layers are not serializable, hence this class is PartiallySerializable and the Tiled components are transient. See WorldTerrain.deserialize.
